🌱 what is PlantPal?

PlantPal is a smart plant care tracker that helps you know exactly which plant needs attention today. Add your plants with their watering frequency, and PlantPal will always show you the most urgent one first — no more forgotten cacti.

Under the hood, PlantPal uses two classic data structures: a min-heap for scheduling and a graph for companion planting recommendations.

data structure 1 — min-heap

A min-heap is a complete binary tree where each node's value is smaller than its children. In PlantPal, each node is a plant, and its value is the number of days until it needs care. The plant with the fewest days remaining is always at the root — instantly accessible.

When you mark a plant as watered, it gets removed from the heap and re-inserted with its next care date. The heap self-organizes in O(log n) time.

operationcomplexitydescription
insertO(log n)add a plant, bubble up to correct position
extract-minO(log n)remove most urgent plant, heapify down
peekO(1)see most urgent plant without removing
build heapO(n)build from existing array of plants

🕸 data structure 2 — graph with BFS

The companion planting network is an undirected graph stored as an adjacency list. Each plant is a node. An edge between two plants means they are compatible companions — they grow well together.

Breadth-First Search (BFS) is used to traverse the graph from any starting plant. BFS visits all plants one connection away first, then two connections away, etc. — revealing the full network of compatible companions in order of closeness.

operationcomplexitydescription
add edgeO(1)add compatibility link between two plants
BFS traversalO(V + E)visit all reachable companions
adjacency lookupO(degree)find all companions of a plant

how to run

PlantPal is a single HTML file with no dependencies. Open plantpal.html in any modern browser. No server, no installation needed.

To add a plant: click + add plant in the top right. Fill in the name, type, emoji, and watering frequency. The plant will be inserted into the min-heap and appear in the care queue immediately.
